# Select & Paste

A tiny Chrome extension: select text to copy it instantly, double-click any editable field to paste. No keyboard shortcuts to remember.

## Features

- **Select to copy** — release your mouse (or finish a keyboard selection) after selecting text and it's copied automatically
- **Double-click to paste** — double-click any input, textarea, or editable field to paste your clipboard at the cursor
- **Works with keyboard selection too** — Shift+Arrow, Ctrl+A, etc. all trigger auto-copy, not just mouse drags
- **Copy confirmation** — a small "Copied" flash confirms it worked
- **Password field safe** — never auto-copies out of a password input
- **One-click toggle** — pause it from the popup without disabling the extension

## Installation

1. Go to `chrome://extensions` (or `brave://extensions`)
2. Enable **Developer mode**
3. Click **Load unpacked** → select this folder

## Note

Double-click normally selects the word under your cursor. This extension overrides that behavior on editable fields (inputs, textareas, contenteditable) to paste instead. Double-click still works normally everywhere else, like selecting a word on a webpage.
